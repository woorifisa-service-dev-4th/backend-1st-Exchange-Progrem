
## 서비스 소개
KRW, USD, EUR, JPY, CNY 중 하나를 입력받아 입력받은 통화 외의 통화로 변환하는 프로그램입니다 


## 실행

1. 아래 링크 다운로드 후 압축해제
[Exchange.jar.zip](https://github.com/user-attachments/files/18744703/Exchange.jar.zip)

2. 해당 폴더로 이동후 아래 명령어 실행
       
        java -jar Exchange.jar
